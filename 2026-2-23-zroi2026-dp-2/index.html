决策单调性

定义

考虑函数 f(i)=min⁡1≤i<i{w(j,i)}f(i) = \min\limits_{1 \le i < i}\{w(j, i)\}f(i)=1≤i<imin​{w(j,i)}，这个函数对应 g(i)=arg min⁡1≤i<i{w(j,i)}g(i) = \argmin\limits_{1 \le i < i}\{w(j, i)\}g(i)=1≤i<iargmin​{w(j,i)}，若 g(i)g(i)g(i) 单调递增，则称 www 满足决策单调性。

一个更强的条件是四边形不等式。若 ∀l1<l2<r1<r2,w(l1,r1)+w(l2,r2)≤w(l1,r2)+w(l2,r1)\forall l_1 < l_2 < r_1 < r_2,w(l_1, r_1) + w(l_2, r_2) \le w(l_1, r_2) + w(l_2, r_1)∀l1​<l2​<r1​<r2​,w(l1​,r1​)+w(l2​,r2​)≤w(l1​,r2​)+w(l2​,r1​)，则称 www 满足四边形不等式。

一个结论是：若 i<j<l<r,w(i,l)≤w(j,l)i < j < l < r, w(i, l) \le w(j, l)i<j<l<r,w(i,l)≤w(j,l)，则 w(i,r)≤w(j,r)w(i, r) \le w(j, r)w(i,r)≤w(j,r)。

对于一个满足决策单调性的 www，其必然满足决策单调性。

处理

计算一个这样的问题有两个方法：二分队列和分治。

二分队列

由于单调性的存在，每个位置一定对后面一个连续区间产生贡献。维护一个队列，统计每一个 iii 能够成为最优决策的区间。当出现一个新的位置可用于贡献的时候，由于它是最后加入的，根据单调性它一定对一段后缀产生贡献。因此只需要找出最左边的能够产生贡献的节点即可。复杂度 O(nlog⁡n)O(n \log n)O(nlogn)。

斜率优化本质上就是采用了这种方法（当然某些特殊情况下可以省去二分，改用单调队列）。这种做法的优点是可以半在线地处理问题，并且可以做到线性空间复杂度。但是缺点是 www 必须能够 O(1)O(1)O(1) 计算，否则二分就挂了。

分治

这个方法不考虑前面对后面的贡献，而是直接计算。假设要计算一个区间 [l,r][l,r][l,r]的 fff，已知这些 fff 的最优决策点属于区间 [L,R][L, R][L,R]。

如果 L=RL=RL=R，则这些 fff 的决策均已知，可以计算。否则，首先设中点 mid=l+r2mid=\frac{l+r}2mid=2l+r​，暴力求出 f(mid)f(mid)f(mid) 的最优决策点 PPP，那么 [l,mid−1][l,mid - 1][l,mid−1] 的可能决策点就在 [L,P][L, P][L,P] 上，而 [mid+1,r][mid + 1, r][mid+1,r] 的可能决策点在 [P,R][P, R][P,R] 上。这样复杂度 O(nlog⁡n)O(n \log n)O(nlogn)。

它的优点是同一层的 www 可以较为复杂，缺点就是没法解决半在线问题。

[JSOI2016] 灯塔

不难发现可以移项，得到 pi=max⁡j{hj−hi+∣i−j∣}p_i = \max\limits_j\{h_j - h_i + \sqrt{|i - j|}\}pi​=jmax​{hj​−hi​+∣i−j∣​}。

j<ij < ij<i 和 j>ij > ij>i 是两种无关的情况，分别计算，这样就去掉了绝对值。这里只考虑 j<ij < ij<i 的情况，有 w(j,i)=hj−hi+i−jw(j, i) = h_j - h_i + \sqrt{i - j}w(j,i)=hj​−hi​+i−j​。这个东西是满足四边形不等式的，因为一次项是全都相等的，f(x)=x+(k−x)f(x) = \sqrt x + \sqrt{(k - x)}f(x)=x​+(k−x)​ 在 x=k−xx = k - xx=k−x 时有最大值（证明可以求导或者均值，不过感性理解一下还是挺对的）。这样就可以跑决策单调性了。

[NOI2009] 诗人小G

f(i)=min⁡1≤j≤i{f(j−1)+∣si−sj−L−1∣P} f(i) = \min\limits_{1 \le j \le i}\{f(j - 1) + |s_i - s_j - L - 1|^P\} f(i)=1≤j≤imin​{f(j−1)+∣si​−sj​−L−1∣P}

同上面一样，后面这一坨满足四边形不等式，证明也用差不多的办法。不过这个东西必须二分队列。

任务分配问题

也可以简单写出一个 DP：

f(i)=min⁡1≤j≤i{f(j−1)+w(j,i)} f(i) = \min\limits_{1 \le j \le i}\{f(j - 1) + w(j, i)\} f(i)=1≤j≤imin​{f(j−1)+w(j,i)}

其中 www 表示区间内的顺序对数量。不难发现 www 仍然是一个满足四边形不等式的东西，因为大区间内会包含左右两个区间内的所有顺序对，对于中间部分小的区间会再算一次，这些部分是平的，而大区间挖掉小区间的部分的顺序对是多出来的，因此要大。

但是这没完，因为 www 没法 O(1)O(1)O(1) 计算。二分队列看起来不大能优化，而分治由于一次要算一层所以看起来优化空间比较大。实际上，www 可以双指针计算。

分治的问题是没法半在线，解决这个问题得用点科技。

考虑改变分治的过程，假设现在已知 lll 的最优点 LLL 和 rrr 在 [1,l−1][1, l - 1][1,l−1] 中的最优点 RRR，还是计算 midmidmid 在 [1,l−1][1, l - 1][1,l−1] 上的最优决策点。这样左半边就可以递归了，递归完了以后再更新 [mid,r][mid, r][mid,r] 的最优决策点，发现右边也可以递归了，过程结束。

环上排序信息最优分割

首先有个暴力的 DP：先枚举第一个序列的切断位置，然后每次跑一个 O(M2)O(M^2)O(M2) 的 DP，总复杂度 O(M3)O(M^3)O(M3)。内层的 DP 应该长这个样子：

f(k,i)=min⁡j{f(k−1,j−1)+w(k,j,i)} f(k, i) = \min_j\{f(k - 1, j - 1) + w(k, j, i)\} f(k,i)=jmin​{f(k−1,j−1)+w(k,j,i)}

其中 f(k,i)f(k, i)f(k,i) 表示前 kkk 个区间，当前区间在 iii 砍；w(k,j,i)w(k, j, i)w(k,j,i) 表示 k−1k - 1k−1 个区间的 jjj 以后部分和 kkk 区间 iii 以前的部分拼起来的代价。这个可以直接预处理。

当然，要考虑如何优化。kkk 可以滚掉，那么假设 kkk 固定，可以发现 w(j,i)w(j, i)w(j,i) 是满足四边形不等式的。

证明起来似乎有点困难，不过可以打表。一个可行的证明长这个样子：

我们证明函数 f(l,r)f(l, r)f(l,r) 满足四边形不等式，即对于任意 l1≤l2≤r1≤r2l_1 \le l_2 \le r_1 \le r_2l1​≤l2​≤r1​≤r2​，有

>f(l1,r1)+f(l2,r2)≤f(l1,r2)+f(l2,r1).> > f(l_1, r_1) + f(l_2, r_2) \le f(l_1, r_2) + f(l_2, r_1). > >f(l1​,r1​)+f(l2​,r2​)≤f(l1​,r2​)+f(l2​,r1​).>

设 A=S(l1,r1)A = S(l_1, r_1)A=S(l1​,r1​)，B=S(l2,r2)B = S(l_2, r_2)B=S(l2​,r2​)，则 A∪B=S(l1,r2)A \cup B = S(l_1, r_2)A∪B=S(l1​,r2​)，A∩B=S(l2,r1)A \cap B = S(l_2, r_1)A∩B=S(l2​,r1​)。我们需要证明

>f(A)+f(B)≤f(A∪B)+f(A∩B).> > f(A) + f(B) \le f(A \cup B) + f(A \cap B). > >f(A)+f(B)≤f(A∪B)+f(A∩B).>

考虑 g(S)=−f(S)g(S) = -f(S)g(S)=−f(S)，则 ggg 是递增的（添加点使 fff 减小，ggg 增大）。下面证明 ggg 具有次模性，即对任意两个集合 X,YX, YX,Y 有

>g(X)+g(Y)≥g(X∪Y)+g(X∩Y),> > g(X) + g(Y) \ge g(X \cup Y) + g(X \cap Y), > >g(X)+g(Y)≥g(X∪Y)+g(X∩Y),>

这等价于 fff 的超模性。

次模性可由边际收益递减推出：对于任意 S⊆TS \subseteq TS⊆T 和任意点 x∉Tx \notin Tx∈/T，有

>g(S∪{x})−g(S)≥g(T∪{x})−g(T).> > g(S \cup \{x\}) - g(S) \ge g(T \cup \{x\}) - g(T). > >g(S∪{x})−g(S)≥g(T∪{x})−g(T).>

计算边际收益。设 SSS 中排序后 xxx 的左右邻居为 ℓS\ell_SℓS​ 和 rSr_SrS​（包括端点 000 和 2×1062\times 10^62×106），则插入 xxx 后，fff 的变化为

>f(S)−f(S∪{x})=(rS−ℓS)2−[(x−ℓS)2+(rS−x)2]=2(rS−x)(x−ℓS).> > f(S) - f(S \cup \{x\}) = (r_S - \ell_S)^2 - [(x - \ell_S)^2 + (r_S - x)^2] = 2(r_S - x)(x - \ell_S). > >f(S)−f(S∪{x})=(rS​−ℓS​)2−[(x−ℓS​)2+(rS​−x)2]=2(rS​−x)(x−ℓS​).>

>g(S∪{x})−g(S)=f(S)−f(S∪{x})=2(rS−x)(x−ℓS).> > g(S \cup \{x\}) - g(S) = f(S) - f(S \cup \{x\}) = 2(r_S - x)(x - \ell_S). > >g(S∪{x})−g(S)=f(S)−f(S∪{x})=2(rS​−x)(x−ℓS​).>

>g(S∪{x})−g(S)≥g(T∪{x})−g(T).> > g(S \cup \{x\}) - g(S) \ge g(T \cup \{x\}) - g(T). > >g(S∪{x})−g(S)≥g(T∪{x})−g(T).>

>−f(A)−f(B)≥−f(A∪B)−f(A∩B),> > -f(A) - f(B) \ge -f(A \cup B) - f(A \cap B), > >−f(A)−f(B)≥−f(A∪B)−f(A∩B),>

>f(A)+f(B)≤f(A∪B)+f(A∩B).> > f(A) + f(B) \le f(A \cup B) + f(A \cap B). > >f(A)+f(B)≤f(A∪B)+f(A∩B).>

代入 A,BA, BA,B 即得所证。故 fff 满足四边形不等式。

但是这还是不大能过。最大的卡点在于还是要枚举第一刀的位置，这很浪费时间。假如在第一个序列的 iii 位置切了一刀，找出此时每个序列的最优决策点，然后再改为在 i+1i + 1i+1 位置切一刀，感性理解一下应该会发现每个最优决策点应该都不会左移。证明可以反证。

也就是说外层也是个满足决策单调性的东西。这样，外层再套一个分治就可以做到 O(Mlog⁡3M)O(M \log^3 M)O(Mlog3M)。已经比较优秀了。两个分治都不像是能优化了，只有 set 看起来还有一定的优化空间。实际上，如果仅仅是插入和删除操作的话，双向链表是比较好的选择。由于整个过程中，可以把一种操作变成撤销，删除和撤销操作的复杂度都是 O(1)O(1)O(1) 的，故复杂度压掉一个 log⁡\loglog。

[USACO19FEB] Mowing Mischief P

首先第一个限制是 LIS，先求出来。

不难发现对于某种路径，一定是一个先向右再向下，另一个相反，这样围成一个矩形。因此问题就变成了最小化相邻两个点围起来的矩形的面积和。因此要计算这个东西应当是按照以 iii 为最后一个节点的 LIS 长度划分阶段。

这样就可以初步写出一个方程：

f(k,i)=min⁡j{f(k−1,j)+w(k,j,i)} f(k,i) = \min_j\{f(k - 1, j) + w(k, j, i)\} f(k,i)=jmin​{f(k−1,j)+w(k,j,i)}

其中 kkk 表示阶段，iii 表示当前阶段的第 iii 个点，要求 jjj 满足其 LIS 为 k−1k - 1k−1 且能够转移到 iii。这里可以令第 kkk 阶段的第 iii 个点为 BBB，转移过来的节点是 AAA。那么 xA≤xBx_A \le x_BxA​≤xB​ 且 yA≤yBy_A \le y_ByA​≤yB​。www 表示转移代价，也就是矩形大小，应当为 (xB−xA)(yB−yA)(x_B - x_A)(y_B - y_A)(xB​−xA​)(yB​−yA​)。

一个重要的观察是，如果将同一层的点按照 xxx 升序排序，那么 yyy 应当是降序的。证明是简单的，如果不是这一层就能转移了，LIS 不应该相等。

有了这个以后，研究 www 是否满足四边形不等式。设在第 d−1d-1d−1 层有两个点 A1,A2A_1, A_2A1​,A2​（A1A_1A1​ 在 A2A_2A2​ 左侧，即 xA1<xA2,yA1>yA2x_{A_1} < x_{A_2}, y_{A_1} > y_{A_2}xA1​​<xA2​​,yA1​​>yA2​​），在第 ddd 层有两个点 B1,B2B_1, B_2B1​,B2​（B1B_1B1​ 在 B2B_2B2​ 左侧，即 xB1<xB2,yB1>yB2x_{B_1} < x_{B_2}, y_{B_1} > y_{B_2}xB1​​<xB2​​,yB1​​>yB2​​）。

计算可以作差：

Δ=[w(A1,B1)+w(A2,B2)]−[w(A1,B2)+w(A2,B1)] \Delta =[w(A_1, B_1) + w(A_2, B_2)] -[w(A_1, B_2) + w(A_2, B_1)] Δ=[w(A1​,B1​)+w(A2​,B2​)]−[w(A1​,B2​)+w(A2​,B1​)]

化简后得到这样的东西：

Δ=(xB2−xB1)(yA1−yA2)+(xA2−xA1)(yB1−yB2) \Delta = (x_{B_2} - x_{B_1})(y_{A_1} - y_{A_2}) + (x_{A_2} - x_{A_1})(y_{B_1} - y_{B_2}) Δ=(xB2​​−xB1​​)(yA1​​−yA2​​)+(xA2​​−xA1​​)(yB1​​−yB2​​)

根据基本假设这些差值全是正的，于是发现这个东西满足反过来的四边形不等式。这意味着其决策是单调递减的。

这看起来很决策单调性，但是实际上有个问题：由于当 xBx_BxB​ 增大的时候，其可以转移来的 AAA 在右移，也就是说，假如 A(3,4)A(3,4)A(3,4) 是 B1(4,6)B_1(4,6)B1​(4,6) 的最优决策点，那么对于一个 B2(7,3)B_2(7,3)B2​(7,3)，这个 AAA 就不再是合法转移了（是不合法不是不优）。也就是说这个东西某种意义上不满足决策单调性，因此不能简单地跑分治板子。

解决方案是利用一个类似于线段树的结构，将一个限制区间给干掉。对一个 BBB，其合法的 AAA 一定是一段连续的区间，那么用线段树将这个区间分成若干个区间，最后对这些区间的答案取个 min⁡\minmin 就可以了。对于对于一个区间内部，所有的节点的地位都是相同的，不存在限制的区别，因此可以跑普通分治或者二分队列。复杂度 O(Nlog⁡2n)O(N \log^2 n)O(Nlog2n)。
